Dataset Description

The dataset consists of labeled images of road surface defects.

Classes:
- Pothole
- Manhole
- Speedbreaker

Annotation Format: YOLOv8
Image Size: 640x640
Split Ratio:
- Training: 60%
- Validation: 20%
- Testing: 20%

Data augmentation techniques were applied to improve robustness.
